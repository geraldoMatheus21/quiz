# 🎯 Quiz Criativo

**Quiz Criativo** é uma plataforma web para criação e execução de quizzes educacionais personalizáveis. Professores podem criar quizzes com perguntas de múltipla escolha, temas visuais e imagens. Alunos acessam todos os quizzes disponíveis, respondem e acompanham estatísticas de acertos.

## 🚀 Tecnologias

- **Frontend:** React + Vite, Tailwind CSS, React Router DOM
- **Backend & Banco de Dados:** Firebase (Authentication, Firestore, Storage)
- **Hospedagem:** Vercel (planejada)

## ✨ Funcionalidades

### 👨‍🏫 Professor
- Cadastro/login com nome, e-mail e senha
- Criar quizzes (título, descrição, tema visual)
- Adicionar perguntas com alternativas e marcar a correta
- Visualizar todos os quizzes criados
- Código de acesso gerado automaticamente

### 🧑‍🎓 Aluno
- Cadastro/login com nome, e-mail e senha
- Listar todos os quizzes disponíveis (sem necessidade de código)
- Responder perguntas uma a uma
- Ver estatísticas: melhor percentual de acerto por quiz e número de tentativas
